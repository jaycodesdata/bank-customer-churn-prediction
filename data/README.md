# Dataset Information

## Dataset Source
This project uses the **Bank Customer Churn Dataset**, which is publicly available on Kaggle:

https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn/data

## Usage Instructions
The dataset is not included in this repository due to licensing and repository size considerations.

To run the notebook:

1. Download the dataset from Kaggle using the link above.
2. Place the dataset file inside this `data/` folder.
3. Ensure the file path matches the notebook before running the project.
